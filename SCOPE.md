# Project Scope

## In Scope
- Core expense tracking (equal, unequal, percentage, share splits).
- Group management and shareable link invitations.
- Settlement tracking between members.
- Importing legacy CSV data with interactive anomaly resolution.
- Multi-currency support (e.g., INR, USD) within the same group (if approved) or forcing exchange rates at import.
- Guest user handling (e.g., "Dev's friend Kabir").

## Out of Scope
- Automatic currency conversion using live exchange rates.
- Silent autocorrection of critical financial data during CSV import (strict interactive resolution).
